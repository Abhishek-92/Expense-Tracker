# Expense-Tracker

Created a RESTful web service for the following users:

#Manager:

	  	Login / logout
		Change password
		Add expenses
		Update expenses
		Delete expenses
		Can check all the details approved by HR

#HR:

		Login / logout
		Change password
		Verify new expenses
		Approve or reject the expenses
		Check all the approved and rejected expenses

#Admin:

		Login / logout
		Change password
		Check verified expenses by HR
		Calculate the final budget
